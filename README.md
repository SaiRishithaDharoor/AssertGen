# AssertGen

Automatic SystemVerilog Assertion Recommendation Engine

AssertGen is an open-source static analysis tool for SystemVerilog RTL.

It analyzes RTL designs, detects common hardware structures such as:

- Counters
- FSMs
- FIFOs
- Handshake protocols
- Arbiters

and recommends useful SystemVerilog Assertions (SVA) that can serve as a starting point for formal verification.

## Features

- RTL Parsing
- Hardware Pattern Detection
- Assertion Recommendation
- HTML Report Generation
